The Automatic Blogroll Widget allows you to display an automatically updated blogroll in your sidebar!

== Description ==

Here is the second of our newest widgets - The Automatic Blogroll. It allows you to display an automatically updated blogroll in your sidebar.
This blogroll will display the blogs that quote you the most, or those that you quote the most, or both!
The simplified interface will allow you to personalize all your Wikio tools without any technical know-how at all.

Important: This plugin is still in BETA TESTING.


== Installation ==

1. Upload the Wikio pack into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What languages are available? =

English, french, Italian, Spanish, German. Some .po files are available for your own translations.

= Content =
Wikio Share, Vote and Subscribe buttons, Top blog badge, and a news widget.
